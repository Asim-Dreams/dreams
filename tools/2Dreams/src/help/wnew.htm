What's new?

Tabed Item Window

With version 1.3.0 tab support for Item Windows was added. You can have multiple views of the same information and split by different
criteria. The information to show is specified in the ADF (xml) configuration file.



Bug Fixing

 

A bunch of minor bugs have been fixed


Performance

 Some non trivial optimizations have been done in the code related with file loading and screen refresh.
